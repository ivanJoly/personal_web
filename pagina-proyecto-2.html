Baguen Propiedades

Plataforma auto-administrable inmobiliaria

https://baguendesarrollos.com.ar

Página web con panel de administración de propiedades, buscador con filtro completo, carrusel de propiedades destacada, propiedades relacionadas y mapa de propiedades interactivo. Tecnología Wordpress con implementaciones PHP y JS

Este sitio fue realizado tomando como punto de partida el Theme de Wordpress My Home y configurandolo exhaustivamente, a través de implementaciones en PHP y JS, hasta lograr el objetivo deseado. Para facilitarle al cliente la carga de propiedades, se tomo como base el sistema de zonaprop (que el cliente ya conocia y utilizaba hace muchos años). Se resolvieron cuestiones de naturaleza tanto front como back-end para generar una adaptación del sistema alcanzando lo requerido por la inmobiliaria.

RESOLVIENDO UN PROBLEMA

Acostumbrados a vender y alquilar propiedades a través de páginas como Zonaprop la empresa notó que al compartir links de sus propiedades con posibles clientes perdían muchos en manos de otras inmobiliarias a través del apartado de propiedades relacionadas que las plataformas brindaban. Se le ofreció un sitio web personal para que los posibles clientes nunca abandonaran el ecosistema propio de la empresa.

ADAPTANDOSÉ AL CLIENTE

El panel de carga de propiedades se hizo basado en la estructura de ZONAPROP (la plataforma que el cliente ya conocía) para facilitar la adaptación del mismo a esta nueva tecnología. Tambien se diseño con un sistema mobile liviano para que el cliente pueda subir propiedades desde su celular sin problemas y en pocos pasos, incluso en situaciones en las que no cuente con WI-FI.
